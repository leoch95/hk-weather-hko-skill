# 使用條款、來源標註與免責（HKO Open Data）

本 skill 使用香港天文台（Hong Kong Observatory, HKO）之開放數據 API。

## 來源標註（Attribution）

如用於任何展示/報告/系統輸出，請保留／加入來源標註：

- 「資料來源：香港天文台」

## 免責聲明（Disclaimer）

- 天氣資料與警告資訊以香港天文台官方最新公布為準。
- 本 skill 只供一般參考用途。
- 不建議用於航空、航海或其他安全關鍵場景。

## 技術特性

- 本 skill 會透過互聯網即時請求 HKO API。
- HKO API schema/欄位可能會調整；如回應格式變更可能導致解析/輸出變更。
